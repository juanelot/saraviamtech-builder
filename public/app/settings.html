Uso local con ngrok
⚠️ Requerido para generación de videos en local

kie.ai necesita una URL pública accesible desde internet para entregar los videos generados. Si corres el servidor en local (localhost), debes usar un túnel como ngrok.

Pasos:
1. Instala ngrok: npm install -g ngrok o descarga desde ngrok.com
2. Expón el puerto: ngrok http 3000
3. Copia la URL HTTPS que te da ngrok (ej: https://abc123.ngrok-free.app)
4. Configura BASE_URL abajo con esa URL
5. Reinicia el servidor

En VPS / producción con dominio propio, no necesitas ngrok — configura BASE_URL con tu dominio.
